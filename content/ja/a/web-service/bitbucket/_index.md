Atlassian社が提供しているソースコードのホスティングサービス。  
JIRAなど他のAtlassian製品との連携が強み（だと思う）。

## 提供形態

- クラウド ... Bitbucket Cloud
- セルフホスト ... オンプレミス
  - Bitbucket Server ... 単一サーバ
  - Bitbucket Datacenter ... クラスタ構成
